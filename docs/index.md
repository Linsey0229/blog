# 欢迎来到我的博客

这里记录我的学习、思考和一切有趣的事。

## 关于我

一个喜欢读书、玩游戏、写点东西的人。

以品析人物塑造为主，做过人物鉴赏视频、写过同人文。有自己原创的oc与世界观，且均有美术形象。

我喜欢看颓废美丽且糜烂的情感，喜欢看两个人在崩坏交错的命运中一起堕落，像野兽一样交合直至生命尽头。

我喜欢看像羽毛一样轻飘又沉重的情感，迎着微风缓缓飘下来的不仅仅是羽毛，还有充满铁锈气息的雨水。

我喜欢看苦涩、不完满而永恒的爱恋。如同枯萎的玫瑰，无法宣泄的爱意以永恒而刺痛的方式存于世间，看到你幸福我是该痛苦还是该祝福？我不知道，但我会永远守望着你，直到世界尽头。
