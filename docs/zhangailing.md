# 《倾城之恋》：一座城的沦陷，成全了白流苏的爱情

**但这真的是她想要的吗？**

如果不是突如其来的战火，那么白流苏一生也无法嫁给范柳原，她会成为范柳原的情妇，在空旷的大房子里过完终日不见光的作为金丝雀的一生。

就在白流苏豁出去要当范柳原情妇的时刻，香港沦陷了。

一方人在死去，一方人在相爱。

在战火导致的一无所有中，范柳原终于愿意娶白流苏了。宏大的战争消解了人类所有的矛盾与价值，使他们步入了婚姻的殿堂。

可是现在战争结束了，他们又要回到人类社会中去。平淡如水的婚姻真的是白流苏想要的吗？她得到了长期饭票，也失去了曾经在爱情博弈中偶然窥见的真心与心动。

她成为了范柳原的妻子，范柳原再也没和她说过那些情话。
